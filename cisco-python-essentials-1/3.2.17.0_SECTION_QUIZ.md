**Question 1:** Create a for loop that counts from 0 to 10, and prints odd numbers to the screen. Use the skeleton below:

```
for i in range(1, 11):
    # Line of code.
        # Line of code.
```

<details>
  <summary>Sample Solution</summary>

  ```python
  for i in range(0, 11):
      if i % 2 != 0:
          print(i)
  ```
